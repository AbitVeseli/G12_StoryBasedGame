Denna kod är skriven i JavaFX på Intellij För att starta programmet så klicka dig fram till "Main".

Filväg i Intellij : src -> main -> java -> org.exemple.g12_storybasedgame -> controller -> Main

Om delar av spelet inte syns på skärmen så kan du behöva ändra skärmstorlek på spelet eller din dator. Rekomenderat är att inte ha i helskärmsläge. 

Skapa ett konto för att logga in.

När du loggat in är det bara att utforska produkten.
